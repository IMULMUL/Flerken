Scalability

Flerken supports cross-platform obfuscation detection. Both Windows and Linux command line stings can be analyzed to check if there is any obfuscation.

Platform Recognition

We provide a default setting if you lack of information about which platform the tested string belongs to. Flerken can automatically recognize the likely platform and present this information in the results.

Accuracy

Flerken is adequate to distinguish most Windows/Linux command obfuscations. The effectiveness of Flerken is evaluated via representative black/white command samples.
